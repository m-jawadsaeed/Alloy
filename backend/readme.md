# TaskFlow Backend

## Stack
- Express
- TypeScript
- Prisma
- PostgreSQL
- Redis
- Socket.io

## Environment

```env
PORT=5000
DATABASE_URL=postgresql://postgres:postgres@localhost:5432/taskflow
CLIENT_URL=http://localhost:5173
JWT_ACCESS_SECRET=your_access_secret
JWT_REFRESH_SECRET=your_refresh_secret
REDIS_HOST=localhost
REDIS_PORT=6379
```
